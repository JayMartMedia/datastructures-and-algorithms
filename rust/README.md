# Rust

## Requirements
- Rust and Cargo installed

## Running Rust Snippets
Use `cargo test --example {name of snippet}` to run tests for a specific snippet, where {name of snippet} is the name of the file as listed in the examples section of the [Cargo.toml file](./Cargo.toml)

For example, to run the binary search snippet, run `cargo test --example binary-search`
